    Step 4. Download NYC open data and upload to QGIS :

    • A. Visit New York City Open Data website and search for GIS files.



    • B. Peruse the files for a moment to see what is available for download. Search for "borough boundaries" and go to the website.



    • C. Export the layer as a shapefile and a geojson file


    • D. After you download and install QGIS, open the program and drag the shapefile into the viewing area. Save your QGIS project so you can continue working on it later.
