Creating a Map of the New York City Tri-State Area

This code defines and configures a map of the New York City (NYC) Tri-State area using the Leaflet library. It sets specific bounds for the map, ensuring that users cannot zoom out beyond a certain level or pan beyond the defined area. The map is centered at the coordinates of New York City with an initial zoom level of 12.


            <!-- Define the bounds for the New York City Tri-State area -->
            var bounds = L.latLngBounds(
                L.latLng(40.4774, -74.2591), // Southwest corner (bottom-left)
                L.latLng(41.4754, -73.3913)  // Northeast corner (top-right)
            );
            
            <!-- Create and configure the map with the specified bounds -->
            var map = L.map('nymap', {
                maxBounds: bounds,          // Set maxBounds to limit zooming out
                maxBoundsViscosity: 1.0,   // Elastic effect on exceeding bounds
                minZoom: 7,                // Minimum zoom level
                maxZoom: 16                // Maximum zoom level (adjust as needed)
            }).setView([40.7128, -74.0060], 12); // New York City coordinates, closer zoom level

A scale control is added to the map to provide users with information about distances on the map.

Marker Size Based on Zoom Level

The code defines a function called calculateMarkerSize(zoom) that calculates the size of markers on the map based on the current zoom level. The marker size decreases as the zoom level increases, ensuring that markers don't become too large when zoomed in.

NYC Counties Layer

This layer represents the counties in the NYC area. It loads GeoJSON data and styles it with gray fill colors, black borders, and a semi-transparent appearance. Additionally, it adds label markers to display county names.


            var nyccountiesLayer = L.geoJSON.ajax('https://aurashak.github.io/geojson/nyc/nyccounties.geojson', {
                style: function (feature) {
                    return {
                        fillColor: 'grey',
                        color: 'black',
                        weight: 0.5,
                        opacity: 0.5,
                        fillOpacity: .8
                    };
                },
                pointToLayer: function (feature, latlng) {
                    // Get the county name from the 'NAME' property
                    var countyName = feature.properties.NAME;
            
                    // Create a label marker with the county name as the label
                    return L.marker(latlng, {
                        icon: L.divIcon({
                            className: 'leaflet-div-label',
                            html: countyName, // Use the 'NAME' property as the label
                            iconSize: [100, 40] // Adjust the size of the label marker
                        })
                    });
                }
            });

Base Map Layers

This section defines base map layers including OpenStreetMap and a satellite layer. Users can switch between these base maps using a control.

Layer Control

A layer control is added to the map to allow users to toggle different layers on and off. It includes base maps and the previously defined NYC Counties Layer.

Layer Groups for Environmental Data

The code defines layer groups for various environmental data categories: AQI sites, water-related layers (100 Year Floodplain and NYC CSO), energy-related layers (Major Oil Storage, Power Plants, NY Gas Pipelines), and waste-related layers (Waste Transfer Facility, Wastewater Treatment, Inactive Solid Waste Landfill, Recycling Facility, Chemical Storage).

Toggle Buttons for Layer Groups

The code includes event listeners for toggle buttons that allow users to show/hide specific layers or groups of layers on the map.

Legend Symbols

Finally, the code includes a function setLegendSymbol to set legend symbols for each layer or group of layers. These symbols represent the layers on the legend with different shapes and colors.
