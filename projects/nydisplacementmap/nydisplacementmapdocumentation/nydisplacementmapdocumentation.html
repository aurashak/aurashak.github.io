NY Displacement Map Documentation

1. Technology and Applications

Downlod QGIS and Photoshop .

2. Sources of data

Download the following files, in shapefile format, from these websites"
  • From https://data.gis.ny.gov, download New York State county boundaries and New York State streets.
  • From https://opendata.cityofnewyork.us, download NYC borough boundaries, NYC streets, NYC hurricane evacuation zones, and NYC projected floodplains.
  • From https://www.geofabrik.de/data/shapefiles.html, download New Jersey and New York state roads and state boundaries.
  • From https://www.geofabrik.de/data/shapefiles.html, download data for per capita income and average cost of housing.


3. Import shapefiles into QGIS

  • Open QGIS and import shapefiles
  • Adjust map projection for preferred perspective
  • Export individual layers with minimal styling and stark contrasting colors. Export outlines with no fill as separate layers. Export layers with the exact same scale and placement on page, use .png file extension and set a transparent backgrounds.
