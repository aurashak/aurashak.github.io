NY Displacement Map Documentation

1. Technology and Applications



This map was created with QGIS and Photoshop software applications.

2. Sources of data

Download shapefile format from these websites

  • NYS GIS Clearinghouse: New York State county boundaries, New York State streets

  • New York City Open Data: New York City borough boundaries, New York City streets, New York City hurricane evacuation zones

  • New Jersey Open Data: NJ State boundary

  • Geofabrik: NJ State and New York state roads and state boundaries

  • US Census: Per capita income data, average coost of housing data

    4. Import individual layers to Photoshop

    Open Photoshop and load the .png layers exported from QGIS
